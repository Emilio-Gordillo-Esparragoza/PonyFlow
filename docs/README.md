# PonyFlow docs

Architecture notes and architecture decision records (ADRs) for this repository.

- Main product overview: [../README.md](../README.md)
- Current runtime shape: [architecture.md](architecture.md)
- Decisions: [decisions/](decisions/)

## ADR conventions

- Files are numbered: `NNNN-short-slug.md`
- Each ADR includes: **Status**, **Context**, **Decision**, **Consequences**
- Status values: `Accepted` | `Superseded` | `Proposed`

When you make a lasting design choice (routing, packaging, protocol), add a new ADR and link it from this index.
